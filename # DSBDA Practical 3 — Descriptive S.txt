# Complete Practical Code

```python
import pandas as pd
import numpy as np

# Titanic Dataset

df = pd.read_csv('titanic.csv')

# Display first rows
print(df.head())

# Group statistics
print(df.groupby('Sex')['Age'].describe())

# Aggregated statistics
print(df.groupby('Sex')['Age'].agg(['mean','median','min','max','std']))

# Lists by category
male_age = df[df['Sex']=='male']['Age'].tolist()
female_age = df[df['Sex']=='female']['Age'].tolist()

print(male_age)
print(female_age)

# Iris Dataset

iris = pd.read_csv('iris.csv')

# Display first rows
print(iris.head())

# Species filtering
setosa = iris[iris['species']=='Iris-setosa']
versicolor = iris[iris['species']=='Iris-versicolor']
virginica = iris[iris['species']=='Iris-virginica']

# Statistics
print('Setosa Statistics')
print(setosa.describe())

print('Versicolor Statistics')
print(versicolor.describe())

print('Virginica Statistics')
print(virginica.describe())

# Mean and Standard Deviation
print(setosa.mean(numeric_only=True))
print(setosa.std(numeric_only=True))
```

# Measures of Central Tendency

## Mean

Average value.

Formula:

```python
Mean = Sum of values / Total values
```

---

## Median

Middle value after sorting.

---

## Mode

Most repeated value.

---

# Measures of Variability

## Standard Deviation

Measures spread of data.

---

## Variance

Square of standard deviation.

---

## Range
