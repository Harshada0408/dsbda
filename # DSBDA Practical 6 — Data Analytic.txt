# Machine Learning Concept

## Naive Bayes

Naive Bayes is a probability-based classification algorithm.

Examples:

* Email Spam Detection
* Disease Prediction
* Flower Classification

### Bayes Theorem Formula

[
P(A|B)=\frac{P(B|A)P(A)}{P(B)}
]

Where:

* P(A|B) → Probability of A given B
* P(B|A) → Probability of B given A
* P(A) → Prior probability
* P(B) → Evidence probability

# Complete Practical Code

```python
import pandas as pd
import numpy as np
from sklearn.model_selection import train_test_split
from sklearn.naive_bayes import GaussianNB
from sklearn.metrics import confusion_matrix, accuracy_score

# Load dataset

df = pd.read_csv('iris.csv')

# Features and Target

X = df.drop('species', axis=1)
y = df['species']

# Train-Test Split

X_train, X_test, y_train, y_test = train_test_split(
    X, y, test_size=0.25, random_state=0
)

# Create Model

model = GaussianNB()

# Train Model

model.fit(X_train, y_train)

# Predictions

y_pred = model.predict(X_test)

# Confusion Matrix

cm = confusion_matrix(y_test, y_pred)
print(cm)

# Accuracy

accuracy = accuracy_score(y_test, y_pred)
print('Accuracy:', accuracy)

# Binary Conversion

y_test_binary = (y_test == 'Iris-setosa')
y_pred_binary = (y_pred == 'Iris-setosa')

# Binary Confusion Matrix

binary_cm = confusion_matrix(y_test_binary, y_pred_binary)

# Extract Metrics

TN = binary_cm[0][0]
FP = binary_cm[0][1]
FN = binary_cm[1][0]
TP = binary_cm[1][1]

# Additional Metrics

error_rate = (FP + FN) / (TP + TN + FP + FN)
precision = TP / (TP + FP)
recall = TP / (TP + FN)

# Print Metrics

print('TP:', TP)
print('TN:', TN)
print('FP:', FP)
print('FN:', FN)
print('Error Rate:', error_rate)
print('Precision:', precision)
print('Recall:', recall)
```
