# Complete Practical Code

```python
import nltk
from nltk.tokenize import word_tokenize
from nltk.corpus import stopwords
from nltk.stem import PorterStemmer
from nltk.stem import WordNetLemmatizer
from nltk import pos_tag
from sklearn.feature_extraction.text import TfidfVectorizer

# Download NLTK resources
nltk.download('punkt')
nltk.download('stopwords')
nltk.download('averaged_perceptron_tagger')
nltk.download('wordnet')

# Sample Text
text = "Text analytics is the process of converting unstructured text into meaningful information. Natural Language Processing helps in analyzing text data."

# Tokenization
tokens = word_tokenize(text)
print('Tokens:', tokens)

# POS Tagging
pos = pos_tag(tokens)
print('POS Tags:', pos)

# Stop Words Removal
stop_words = set(stopwords.words('english'))
filtered_words = []

for word in tokens:
    if word.lower() not in stop_words:
        filtered_words.append(word)

print('Filtered Words:', filtered_words)

# Stemming
stemmer = PorterStemmer()
stemmed_words = []

for word in filtered_words:
    stemmed_words.append(stemmer.stem(word))

print('Stemmed Words:', stemmed_words)

# Lemmatization
lemmatizer = WordNetLemmatizer()
lemmatized_words = []

for word in filtered_words:
    lemmatized_words.append(lemmatizer.lemmatize(word))

print('Lemmatized Words:', lemmatized_words)

# TF-IDF Documents
documents = [
    "Text analytics converts text into data",
    "Natural language processing analyzes text",
    "Text data is important in analytics"
]

# TF-IDF
vectorizer = TfidfVectorizer()
X = vectorizer.fit_transform(documents)

# Output
print('Feature Names:', vectorizer.get_feature_names_out())
print('TF-IDF Matrix:')
print(X.toarray())
```

# Important Concepts

## Tokenization

Sentence → Words.

---

## POS Tagging

Grammar labeling.

---

## Stop Words

Common meaningless words.

---

## Stemming

Cuts word endings.

---

## Lemmatization

Meaningful root word.

---

## TF

Term Frequency.

Formula:

[
TF = \frac{Word\ Count}{Total\ Words}
]

---

## IDF

Inverse Document Frequency.

Formula:

[
IDF = \log\left(\frac{Total\ Documents}{Documents\ Containing\ Word}\right)
]

---

## TF-IDF

Importance score.

Formula:

[
TF-IDF = TF \times IDF
]
