# Complete Practical Code

```python
import pandas as pd
import seaborn as sns
import matplotlib.pyplot as plt

# Load Titanic dataset

df = sns.load_dataset('titanic')

# Display dataset
print(df.head())

# Dataset Information
print(df.info())

# Create Figure
plt.figure(figsize=(8,6))

# Create Boxplot
sns.boxplot(x='sex', y='age', hue='survived', data=df)

# Title
plt.title('Titanic Age Distribution by Gender and Survival')

# Axis Labels
plt.xlabel('Gender')
plt.ylabel('Age')

# Display Plot
plt.show()
```

# Understanding Boxplot Components

## Median Line

Middle value.

---

## Box

Represents Interquartile Range (IQR).

---

## Whiskers

Shows normal spread.

---

## Outliers

Displayed as dots.

---

## IQR Formula

[
IQR = Q3 - Q1
]

Where:

* Q1 → 25th percentile
* Q3 → 75th percentile

---
