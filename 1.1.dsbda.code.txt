# cell 10 – handling missing values in age column


# filling missing values in age column using mean

df['Age'].fillna(
    df['Age'].mean(),
    inplace=True
)

# mean is used because age is numerical data

print(df['Age'].isnull().sum())


---

# cell 11 – handling missing values in embarked column


# filling missing values in embarked column using mode

df['Embarked'].fillna(
    df['Embarked'].mode()[0],
    inplace=True
)

# mode is used because embarked is categorical data

print(df['Embarked'].isnull().sum())


---

# cell 12 – handling missing values in cabin column


# checking missing values in cabin column

print(df['Cabin'].isnull().sum())


---

# cell 13 – different ways to handle cabin column


# cabin column has too many missing values

# there are multiple ways to handle missing values

# ---------------------------------------------------
# method 1 -> remove cabin column completely
# ---------------------------------------------------

# this is the most common method because cabin has too many null values

df.drop('Cabin', axis=1, inplace=True)

print(df.head())

# axis=1 means column operation
# inplace=True means permanently modify dataframe


---

# cell 14 – alternative methods for cabin handling


# alternative methods for cabin column

# method 2 -> fill missing values with unknown

# df['Cabin'].fillna('Unknown', inplace=True)

# method 3 -> fill using mode

# df['Cabin'].fillna(df['Cabin'].mode()[0], inplace=True)

# method 4 -> create separate category

# df['Cabin'] = df['Cabin'].fillna('No Cabin')

# note:
# method 1 is preferred here because
# cabin has too many missing values


---

# cell 15 – checking missing values again


# checking missing values after preprocessing

print(df.isnull().sum())


---

# cell 16 – converting data types


# converting age column from float to integer

df['Age'] = df['Age'].astype(int)

# checking updated data types

print(df.dtypes)

# astype() is used for datatype conversion


---

# cell 17 – normalization using min max normalization


# normalizing fare column between 0 and 1

df['Fare_normalized'] = (
    (df['Fare'] - df['Fare'].min()) /
    (df['Fare'].max() - df['Fare'].min())
)

# displaying original and normalized values

print(df[['Fare', 'Fare_normalized']].head())


---

# cell 18 – label encoding sex column


# converting categorical values into numeric values

# male -> 0
# female -> 1

df['Sex'] = df['Sex'].map({
    'male': 0,
    'female': 1
})

# displaying encoded column

print(df[['Sex']].head())


---

# cell 19 – one hot encoding embarked column


# converting embarked categorical column into binary columns

df = pd.get_dummies(
    df,
    columns=['Embarked'],
    dtype=int
)

# dtype=int converts true/false into 1/0

print(df.head())


---

# cell 20 – removing unnecessary columns


# removing unnecessary text columns

# these columns are not useful for machine learning

df.drop(
    ['Name', 'Ticket'],
    axis=1,
    inplace=True
)

# displaying updated dataset

print(df.head())


---

# cell 21 – checking final cleaned dataset


# displaying final dataset information

print(df.info())

# displaying first 5 rows

print(df.head())

* visualization

* noisy

therefore preprocessing is very important.

# important concepts used in practical

---

# dataframe

a dataframe is a table-like data structure provided by pandas.

it contains:

* rows
* columns

similar to:

* excel sheets
* sql tables

---

# missing values

missing values are empty or null values present in dataset.

examples:

* nan
* null
* blank values

missing values can create problems during:

* analysis
* visualization
* machine learning

therefore they must be handled properly.

---

# handling missing values

different methods:

* replacing with mean
* replacing with median
* replacing with mode
* removing rows
* removing columns

in this practical:

* age column missing values were replaced using mean
* embarked column missing values were replaced using mode
* cabin column was removed because it contained too many missing values

---

# datatype conversion

sometimes variables are not stored in correct datatype.

example:

* age may be float instead of integer
* categorical values may be text

datatype conversion helps:

* memory optimization
* proper calculations
* machine learning compatibility

`astype()` function is used for datatype conversion.

---

# normalization

normalization scales data into a fixed range.

mostly:
0 to 1

normalization is important because:

* different features may have different scales
* large values can dominate smaller values
* machine learning models perform better

min-max normalization formula:

x_normalized={x-x_min}/{x_max-x_min}

---

# categorical variables

categorical variables contain categories or text values.

examples:

* male/female
* embarked locations

machine learning algorithms cannot directly understand text values.

therefore categorical data must be converted into numeric form.

---

# label encoding

label encoding converts categories into numeric values.

example:

| category | encoded value |
| -------- | ------------- |
| male     | 0             |
| female   | 1             |

---

# one hot encoding

one hot encoding creates separate binary columns for categories.

example:

| embarked_c | embarked_q | embarked_s |
| ---------- | ---------- | ---------- |
| 1          | 0          | 0          |

this method avoids creating false order relationships between categories.

graphs used:
