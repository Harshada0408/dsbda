WHAT IS LOGISTIC REGRESSION?

MOST IMPORTANT.

Logistic Regression is:

supervised machine learning classification algorithm.

WHAT DOES fit_transform() DO?

Actually TWO steps:

Step	Meaning
fit	learn mean/std

Because:

model must learn ONLY from training data.

ONLY transforms test data.

WHY NOT fit_transform() ON TEST DATA?

Because:

test data should remain unseen.

Otherwise:

data leakage occurs.

MOST IMPORTANT VIVA QUESTION.

WHAT IS DATA LEAKAGE?

Model accidentally learns test information.

Leads to:

unrealistic accuracy.

MOST IMPORTANT PART OF PRACTICAL.

WHAT IS CONFUSION MATRIX?

Table evaluating classification performance.

STRUCTURE
Actual/Predicted	Positive	Negative
Positive	TP	FN
Negative	FP	TN
TRUE POSITIVE (TP)

Model correctly predicts:

positive class.

Example:

customer purchased,
model predicted purchased.
TRUE NEGATIVE (TN)

Model correctly predicts:

negative class.

Example:

customer not purchased,
model predicted not purchased.
FALSE POSITIVE (FP)

Model incorrectly predicts positive.

Example:

customer not purchased,
model predicted purchased.

Also called:

Type I Error
FALSE NEGATIVE (FN)

Model incorrectly predicts negative.

Example:

customer purchased,
model predicted not purchased.

Also called:

Type II Error
