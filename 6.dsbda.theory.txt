Naive Bayes is:

probability-based supervised learning algorithm.

Because it assumes:

WHAT IS BAYES THEOREM?

MOST IMPORTANT FORMULA OF THIS PRACTICAL.

P(A∣B)=
P(B)
P(B∣A)P(A)
	​

P(A)
P(B∣A)
P(B∣¬A)
P(A∣B)=
P(B)
P(B∣A)P(A)
	​

≈0.68,P(B)≈0.25
P(B)=0.25
P(B|A)P(A)=0.17
P(A|B)~0.68
Posterior = useful evidence / total evidence
WHAT EACH TERM MEANS
Symbol	Meaning
P(A|B)	probability of A given B
P(B|A)	probability of B given A
P(A)	probability of A
P(B)	probability of B
SIMPLE REAL UNDERSTANDING

Suppose:

flower petal length is small,
petal width is small.

Then probability of:

Iris-setosa becomes high.

Naive Bayes calculates these probabilities.

WHY GaussianNB() USED?

There are multiple Naive Bayes types.

Type	Used For
GaussianNB	continuous numeric data
MultinomialNB	text/count data
BernoulliNB	binary data

Iris dataset contains:

continuous measurements.

So:

GaussianNB is correct.

MOST IMPORTANT TOPIC.

WHAT IS CONFUSION MATRIX?

Table evaluating classification performance.

STRUCTURE
Actual/Predicted	Positive	Negative
Positive	TP	FN
Negative	FP	TN
TRUE POSITIVE (TP)

Correctly predicted positive.

FALSE POSITIVE (FP)

Incorrectly predicted positive.

Also called:

Type I Error
FALSE NEGATIVE (FN)

Incorrectly predicted negative.

Also called:

Type II Error
TRUE NEGATIVE (TN)

Correctly predicted negative.

CELL 18 — TP FP TN FN FOR EACH CLASS
classes = np.unique(y)

Gets all unique class labels.

WHAT DOES unique() DO?

Returns distinct values.

LOOP
for i, cls in enumerate(classes):

Loops through each class.

WHAT IS enumerate()?

Returns:

index,
value.

Example:

i	cls
0	Setosa
1	Versicolor
TP
TP = conf_mat[i, i]

Diagonal elements.

Correct predictions.

FP
FP = np.sum(conf_mat[:, i]) - TP

Column sum minus TP.

FN
FN = np.sum(conf_mat[i, :]) - TP

Row sum minus TP.

TN
TN = total - (TP+FP+FN)

Remaining correctly rejected samples.
