1. One-Minute Project Explanation

“My mini project is COVID-19 Vaccine Statewise Data Analysis using Python and Pandas.
I performed Exploratory Data Analysis (EDA) on the Indian COVID-19 vaccination dataset obtained from Kaggle.

The dataset contains state-wise vaccination records including first dose, second dose, gender-wise vaccination, vaccine types, sessions, and demographic data.

I used Pandas for data loading, preprocessing, grouping, aggregation, and statistical analysis.

The objectives of my project were:

analyzing first-dose vaccination state-wise,
analyzing second-dose vaccination state-wise,
finding total male and female vaccination counts,
and understanding dataset structure using functions like info() and describe().

I used functions such as groupby(), max(), sum(), info(), and describe() to extract meaningful insights from the dataset.”

2. What Exactly Did You Do?

“First, I loaded the CSV dataset using Pandas.
Then I checked the dataset structure using shape, columns, info(), and describe().
After understanding the data, I performed state-wise analysis using groupby().
Since the vaccination data was cumulative, I used max() to get final vaccination counts for each state.
Then I used sum() for gender-wise analysis.
Finally, I interpreted the results and compared vaccination trends across states.”

5. Important Viva Questions & Answers
What is EDA?

“EDA stands for Exploratory Data Analysis.
It is the process of understanding, analyzing, summarizing, and visualizing data before advanced analytics or machine learning.”

What is a DataFrame?

“A DataFrame is a two-dimensional tabular data structure in Pandas containing rows and columns, similar to an Excel sheet.”

Why did you use groupby()?

“groupby() is used to divide data into groups based on a column.
I used it to analyze vaccination data separately for each state.”

Example:

df.groupby('State')['First Dose Administered'].max()
Why did you use max()?

“The vaccination data is cumulative, meaning values continuously increase over time.
So the maximum value represents the final vaccination count.”

Why not use sum() for first dose?

“Using sum() would repeatedly add cumulative daily values and produce incorrect huge totals.
Since the data is cumulative, max() gives the correct final value.”

What does info() do?

“info() gives:

total rows and columns,

data types,

and memory usage.”
What does describe() do?

“describe() generates statistical summaries such as:

What are missing values?

“Missing values are unavailable or empty entries in a dataset represented as NaN in Pandas.”

How did you handle missing values?

“Pandas aggregation functions automatically ignore NaN values, so explicit handling was not necessary for this analysis.”

Why did you choose this dataset?

“It is a real-world healthcare dataset suitable for learning data analysis and public-health analytics.”

What is the size of your dataset?

What is cumulative data?

“Cumulative data means values continuously increase by adding previous totals.”

Example:

Day 1 → 100

Day 3 → 220
